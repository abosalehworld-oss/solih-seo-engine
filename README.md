## == Description ==
**Solih SEO Engine** is a high-performance, security-hardened internal linking tool. Version 4.0.0 introduces a revolutionary "Zero-Lag" architecture using WP REST API and Transient Caching.

## == Installation ==
1. Upload the folder to `/wp-content/plugins/`.
2. Activate and enjoy the smartest SEO sidebar in WordPress.
